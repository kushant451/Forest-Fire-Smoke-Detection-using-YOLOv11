# Forest Fire and Smoke Detection using YOLOv11

## Overview
This project presents an intelligent real-time forest fire and smoke detection system using the YOLOv11 object detection model. The system is designed to identify fire and smoke in images and video streams at an early stage, enabling rapid response and minimizing environmental damage.

Early detection of forest fires is crucial for preventing large-scale destruction, protecting wildlife, and ensuring human safety. By leveraging deep learning and computer vision, this project provides an automated and efficient monitoring solution.

---

## Features
- Real-time fire and smoke detection
- High accuracy using YOLOv11
- Supports image, video, and webcam input
- Fast inference for practical deployment
- Early warning system for wildfire prevention
- Easy to deploy and customize

---

## Technologies Used

- YOLOv11
